# Portfolio Bio

Uma página pessoal estilo link na bio, desenvolvida com **HTML** e **CSS**, com visual **dark moderno** e foco em apresentação pessoal.

## ✨ Funcionalidades

- Nome de perfil
- Descrição pessoal
- Botões para redes sociais e contato
- Seção de habilidades
- Layout centralizado e responsivo
- Interface moderna com tema escuro

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## 🎨 Estilo visual

O projeto utiliza uma interface com:
- fundo escuro
- destaque em rosa vibrante
- botões modernos
- visual limpo e elegante

## 📚 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- estruturação com HTML
- estilização com CSS
- organização visual
- responsividade básica
- criação de interfaces modernas
